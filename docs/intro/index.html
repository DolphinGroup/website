快速上手

创建应用​

在使用组件之前，需要准备好组件的运行环境。对于新应用而言，可直接在 Febase 上创建 React Native 应用。

按照使用场景的不同，可选 拆包 RN 应用 和 常规 RN 应用 。

安装依赖​

应用准备好后，便可安装相关依赖：

npm i -S @music/dolphin-rn@^1.0.0

然后开发调试：

npm run start

使用组件​

最简单的用法：

import { Button } from '@music/dolphin-rn';

export default () => <Button> hello </Button>;

问题反馈​

如果在开发中遇到了问题，可以到 常见问题 中寻找答案。

对于无法自助解决的，可以在 Playground 上创建一个可复现的最小 Demo，然后在 POPO 群 反馈给我们，我们将在第一时间协助解决问题。
